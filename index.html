多轮对话助手 使用说明 & 注意事项

1. 本助手支持多轮连续对话，自动保存会话历史，AI 可理解上下文内容。

2. 使用前请点击页面右上角「设置API」，输入阿里云百炼 API Key。

3. 左侧侧边栏「新聊天」可创建全新对话，清空历史记忆。

4. 聊天过程中可随时点击「清空对话」重置当前会话记录。

5. 输入框左侧「+」按钮支持上传文件（TXT、MD等文本文件）

6. 注意事项：请勿泄露个人 API Key，密钥仅保存在本地浏览器。
